# Project Paper-Writing Rules

- Lead experimental sections with results, generated figures, and concrete takeaways. Put protocol details in compact paragraphs after the main finding.
- Use confident but precise wording. State evidence boundaries once, then keep result sections focused on what the evaluation shows.
- Every reported number must be traceable to a CSV log, metadata file, or generated output.
- Staged datasets should be framed as reproducible expansion inputs and promoted to main evidence when run logs, budget ledgers, metadata, and figures exist.
